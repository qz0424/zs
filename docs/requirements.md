# 窗帘款式展示系统 - 需求文档

## 1. 产品定位
- **产品名称：** 窗帘款式展示系统
- **目标用户：** 客户（门店顾客、设计师等）
- **使用场景：** 工厂展示、销售员上门演示
- **设备平台：** 手机（安卓 + iOS）

## 2. 核心功能

### 2.1 客户端（客户用）
| 功能 | 说明 | 优先级 |
|------|------|--------|
| 款式展示 | 主图+多色归组，支持视频展示 | P0 |
| 搜索 | 按名称快速搜索 | P0 |
| 筛选 | 按自定义分类维度筛选 | P0 |
| 收藏 | 收藏感兴趣的款式 | P0 |
| 对比 | 把收藏的款式放一起对比 | P0 |

### 2.2 管理端（手机APP）
| 功能 | 说明 | 优先级 |
|------|------|--------|
| 款式管理 | 新增、编辑、下架款式 | P0 |
| 批量导入 | Excel批量导入款式数据 | P0 |
| 图片上传 | 从相册选择或拍照上传 | P0 |
| 视频上传 | 支持视频展示 | P1 |
| 分类管理 | 自定义分类维度和选项 | P0 |

### 2.3 管理后台（电脑网页端）
| 功能 | 说明 | 优先级 |
|------|------|--------|
| 全部管理功能 | 和手机端一样，更方便操作 | P0 |
| 数据统计 | 查看款式数量、上下架情况等 | P1 |

## 3. 数据结构

### 3.1 款式信息
| 字段 | 类型 | 说明 |
|------|------|------|
| id | number | 唯一标识 |
| name | string | 款式名称，如：雅典白纱帘 |
| category | string | 分类（可自定义） |
| material | string | 材质（可自定义） |
| colors | array | 颜色列表（多色归组） |
| style | string | 风格（可自定义） |
| lightLevel | string | 遮光等级（可自定义） |
| mainImages | array | 主图列表 |
| effectImages | array | 效果图列表 |
| video | string | 视频URL |
| sizeRange | string | 尺寸范围 |
| status | string | 状态（上架/下架） |
| createdAt | date | 创建时间 |
| updatedAt | date | 更新时间 |

### 3.2 分类维度
| 维度 | 说明 |
|------|------|
| 适用空间 | 客厅、卧室、书房、办公室、酒店等 |
| 材质 | 棉、麻、涤纶、纱、绒布、遮光布等 |
| 风格 | 现代、欧式、中式、简约、田园等 |
| 遮光等级 | 全遮光、半遮光、不遮光等 |

### 3.3 管理员账户
| 字段 | 类型 | 说明 |
|------|------|------|
| id | number | 唯一标识 |
| username | string | 用户名 |
| password | string | 密码（加密存储） |
| role | string | 角色（admin/editor） |
| createdAt | date | 创建时间 |

## 4. 非功能需求

### 4.1 性能要求
- 首页加载时间 < 3秒
- 图片加载时间 < 2秒
- 支持离线浏览已缓存的款式

### 4.2 安全要求
- 管理员密码加密存储
- API接口认证
- 数据备份

### 4.3 兼容性要求
- 安卓系统：Android 6.0+
- iOS系统：iOS 12.0+
- 屏幕适配：支持不同尺寸手机

## 5. 业务规则

### 5.1 款式管理
- 款式可以上架或下架
- 下架的款式客户不可见
- 支持批量上下架操作

### 5.2 分类管理
- 分类维度可自定义
- 支持添加、编辑、删除分类选项
- 删除分类时需确认是否影响已有款式

### 5.3 图片/视频
- 支持 JPG、PNG 格式图片
- 支持 MP4 格式视频
- 图片自动压缩优化
- 视频限制大小（建议 < 50MB）

## 6. 后续优化功能（第二版）
- 历史记录
- 分享功能
- 收藏夹管理（多个收藏夹）
- 新上架提醒
