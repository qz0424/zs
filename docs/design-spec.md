# 窗帘款式展示系统 - 设计规范

## 1. 设计原则

### 1.1 简洁性
- 界面简洁，突出产品本身
- 减少不必要的装饰元素
- 让用户专注于浏览窗帘款式

### 1.2 易用性
- 操作流程简单直观
- 符合用户习惯
- 减少学习成本

### 1.3 一致性
- 统一的视觉风格
- 统一的交互方式
- 统一的组件规范

## 2. 品牌规范

### 2.1 品牌调性
- 专业、品质、时尚
- 适合家居行业
- 给用户信任感

### 2.2 Logo 规范
- 主Logo：全彩色
- 辅助Logo：单色（黑色/白色）
- 最小尺寸：32x32px

## 3. 色彩规范

### 3.1 主色
```
主色：#2C3E50（深灰色）
用途：标题、重要文字、按钮
```

### 3.2 辅助色
```
辅助色：#E67E22（橙色）
用途：强调、选中状态、操作按钮
```

### 3.3 中性色
```
背景色：#F5F5F5（浅灰色）
卡片背景：#FFFFFF（白色）
分割线：#E0E0E0
次要文字：#999999
```

### 3.4 状态色
```
成功：#27AE60（绿色）
警告：#F39C12（黄色）
错误：#E74C3C（红色）
信息：#3498DB（蓝色）
```

## 4. 字体规范

### 4.1 字体选择
- 中文：思源黑体（Noto Sans SC）
- 英文：Roboto

### 4.2 字号规范
```
标题1：24sp（加粗）
标题2：20sp（加粗）
标题3：18sp（加粗）
正文：16sp
次要文字：14sp
辅助文字：12sp
```

### 4.3 行高规范
- 标题：1.3倍
- 正文：1.5倍

## 5. 间距规范

### 5.1 基础间距
```
xs：4px
sm：8px
md：16px
lg：24px
xl：32px
xxl：48px
```

### 5.2 页面边距
- 左右边距：16px
- 上下边距：24px

### 5.3 卡片间距
- 卡片之间：12px
- 卡片内边距：12px

## 6. 组件规范

### 6.1 按钮
```
高度：48px
圆角：8px
字体：16sp（加粗）
内边距：水平24px
```

### 6.2 输入框
```
高度：48px
圆角：8px
边框：1px solid #E0E0E0
内边距：水平12px
```

### 6.3 卡片
```
圆角：12px
阴影：0 2px 8px rgba(0,0,0,0.1)
背景：白色
```

### 6.4 图片
```
圆角：8px
加载占位：浅灰色背景
加载失败：显示默认图片
```

## 7. 图标规范

### 7.1 图标风格
- 线性图标
- 2px描边
- 圆角端点

### 7.2 图标尺寸
```
小图标：20x20px
中图标：24x24px
大图标：32x32px
```

## 8. 页面规范

### 8.1 首页
```
顶部：搜索栏
中部：分类筛选
主体：款式卡片列表
底部：导航栏（首页/收藏/我的）
```

### 8.2 款式详情页
```
顶部：返回按钮 + 标题
主体：图片轮播 + 视频
中部：款式信息（名称、分类、材质等）
底部：收藏按钮
```

### 8.3 管理端首页
```
顶部：用户信息 + 退出按钮
主体：功能入口（款式管理/分类管理/数据导入）
底部：统计数据
```

## 9. 交互规范

### 9.1 点击反馈
- 按钮点击：颜色变深
- 卡片点击：轻微缩放
- 列表项点击：背景变灰

### 9.2 加载状态
- 页面加载：显示加载动画
- 图片加载：显示占位图
- 提交操作：显示加载中

### 9.3 空状态
- 无数据：显示空状态插图 + 提示文字
- 无搜索结果：显示无结果提示
- 无网络：显示网络错误提示

### 9.4 错误处理
- 网络错误：显示重试按钮
- 服务器错误：显示错误提示
- 操作失败：显示具体错误原因

## 10. 响应式设计

### 10.1 断点设置
```
手机：< 600px
平板：600px - 1024px
电脑：> 1024px
```

### 10.2 布局适配
- 手机：单列布局
- 平板：双列布局
- 电脑：三列布局
