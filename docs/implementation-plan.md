# 窗帘展示系统 - 执行计划（已更新）

## 1. 技术栈（实际采用）

| 层 | 技术 | 说明 |
|----|------|------|
| 服务端 | Node.js + Express | |
| 数据库 | SQLite (better-sqlite3) | 零配置，单文件，适合数百产品 |
| 客户端 | Web SPA (Vanilla JS) | 无构建步骤，可封装为小程序 |
| 管理端 | Web 页面 (Vanilla JS) | 与客户端共用 API |
| 认证 | JWT | 7天过期 |
| 文件上传 | Multer | 格式校验 + 100MB 限制 |

## 3. 关键决策记录
- **Flutter → Web SPA**：零安装、即开即用、后续可封装为小程序
- **PostgreSQL → SQLite**：零配置部署，对本项目规模足够
- **CSV 导入用事务批量插入**：保证数据一致性
- **颜色用 color_name + color_code 表达**：媒体可关联到具体颜色变体
